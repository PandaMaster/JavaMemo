# JavaMemo

Java学习整理

## 数据类型

8种基本类型

| 类型      | size(byte) | 取值范围                |
| --------- | ---------- | ----------------------- |
| int       | 4          | -2147483648～2147483647 |
| short     | 2          | -32768～32767           |
| long      | 8          | -9223372036854775808～9223372036854775807 |
| byte      | 1          | -128～127               |
| float     | 4          | 有效位数为6-7位         |
| double    | 8          | 有效位数为15位          |
| boolean   | 1          | true/false              |
java中没有unsigned类型
